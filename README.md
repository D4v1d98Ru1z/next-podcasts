# App developed in the course of Next.js by Platzi

## Requirements

> Node.JS version 10.7.0
> Npm version 6.1.0

## Install and work
```
git clone https://github.com/rojasleon/next-podcasts.git 
// clone repository
```
```
npm install 
// install all the dependencies
```
```
npm run dev 
// development mode
```
```
npm run build && npm start
// production mode
```
